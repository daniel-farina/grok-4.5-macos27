# macOS 27 Virtual Desktop — Status

**Result: PASS** (static + audit harness)

## App count

**75** official apps registered in `js/apps/registry.js` via `AppRegistry`.

See [APPS.md](./APPS.md) for the full id / name / category table (source of truth regenerated by `node scripts/audit-apps.js`).

## Features

- Liquid Glass desktop chrome (menu bar, dock, Launchpad, Mission Control, Spotlight, Control Center, Notification Center)
- Window manager: drag, resize, focus, minimize, zoom, multi-window z-order
- Appearance: light / dark / auto (`MacShell.setAppearance`)
- Wallpaper cycling (`MacShell.cycleWallpaper`) + Wallpaper app picker
- Stage Manager toggle, hot corners, boot splash
- Dock with 16 apps (finder → trash) + running indicators / bounce
- 75 app UIs (Finder, Safari, Mail, Messages, Settings, Terminal, Calculator, Chess board, Stickies, etc.)
- Desktop widgets (clock, weather, calendar, music)
- Keyboard shortcuts and context menu wallpaper change

## How to open

```bash
# from /Users/web/MacOS
python3 -m http.server 8765
# or: npx serve -l 8765
```

Then open: **http://localhost:8765**
